# User Behavior Funnel Analysis

## Goal
Analyze user behavior and identify drop-off points in the funnel.

## Data
Event-level data: user_id, event_type, timestamp

## Key Findings
- 60% drop at checkout stage
- New users convert worse than returning users

## Tools
- Python
- Pandas
- Matplotlib

## How to run
1. install requirements
2. run notebooks in order
